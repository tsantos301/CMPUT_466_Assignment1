Objects

Objects in code are just like objects in real life. They have properties, values assosciated with them and do certain tasks

In real life a car is an object. A car has properties such as paint color, rim size, vehicle weight, etc. It also has jobs that it performs such driving, braking, starting, stopping, etc.

Objects in code store properties as feilds of the object and tasks that objects perform as functions.

Objects can also been seen as variables except that they can hold many values instead of just one.In JavaScript objects are stored as key value pairs as follows:

var car = {type:"Mercedes", model:"CLA", color:"white"};

"this" Keyword

One of the typically more confusing things in Object Oriented Languages is the "this" keyword.

"this" has different values depending on where it is being used. In general it is used as follows:

  • Inside a method, this refers to the owner object.
  • Alone, this refers to the global object.
  • In a function, this refers to the global object.
