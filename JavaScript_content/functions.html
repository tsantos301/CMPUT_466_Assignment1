Functions

Functions are a block of code that executes a specfic task. Functions can be reused over and over and are executed when they are called.

The basic template for creating a function is as follows:

function functionName(parameter1, parameter2){ return parameter1+parameter2; }

A JavaScript function is created using the function keyword, followed by a name, followed by parentheses (). The parentheses can include parameter names separated by commas: (parameter1, parameter2, ...), however the paranthesis can also be empty if no parameters are needed to be passed into the function. All code which is executed is placed inside curly brackets: {}

During execution if a return statement is reached, the function will stop executing. Functions often compute a return value. The return value is "returned" back to the "caller" but is not required.
