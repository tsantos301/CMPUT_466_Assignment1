Variables

Variables are declared in JavaScript in the same fashion that most programming languages do, and all statements must be followed by a semicolon.

var a, b, c; //This is a comment. a = 10; // A variable a with value 10 b = 5; // A variable b with value 5 c = a - b; // The value of c here is 5

The previous hello world example is actually a perfect example of using these two properties together. When the button is clicked the HTML document is searched for an HTML element which has an ID of "changeText", then the HTML element associated with changeText has its value set to the string Hello World, using the innerHTML property.

<p id = "changeText">Using JavaScript to change HTML content.</p> <button onclick='document.getElementById("changeText").innerHTML = "Hello World!"'>Click to Change Text!</button>
