Hello World:

This first step to take when learning any programming language is to write a basic "hello world" program which prints text to the screen. The code to do so is shown in the code block below, and can be run within the website.

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Hello World.</title> </head> <body> Hello World. </body> </html>
