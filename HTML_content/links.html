Links and Images:

One of the most important HTML5 features is the hyperlink, which references other resources, such as HTML5 documents and images. When a user clicks a hyperlink, the browser tries to execute an action associated with it (for example, navigate to a URL or open an e-mail client). Any displayed element can act as a hyperlink. Web browsers typically underline text hyperlinks and color their text blue by default so that users can distinguish hyperlinks from plain text. Below is an example of creating hyperlinks:

<!DOCTYPE html> <html> <head> <meta charset = "utf-8"> <title> Exercise 2.8 </title> </head> <body> <h1> My Favorite Daily Deals </h1> <ul> <li><a href = "http://www.amazon.ca"> Amazon</a></li> <li><a href = "http://www.ebay.ca"> eBay </a> </li> <li><a href = "http://www.aliexpress.com"> Aliexpress </a></li> </ul> <body/> <html/> Running the above code gives us:

Images:

To simply display an image in an HTML document, one must find a suitable image and place it in the same directory as the html source file. Once this is complete the image can be emebedded in the page using the following:

<img src="image_name.image_extension" alt="Text to use as a placeholder incase the image is not found.">

To adjust the width, height or style of an image one simply uses the following attributes:

<img src="../HTML_examples/Apple.png" alt="Apple Classic Icon" width="128px" height="128px">
