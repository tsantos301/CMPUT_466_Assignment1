Forms:

The HTML form element defines a form that is used to collect user input: An HTML form contains form elements. Form elements are different types of input elements, like text fields, checkboxes, radio buttons, submit buttons, and more.

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Forms.</title> </head> <body> <form> <input type="radio" name="gender" value="male" checked> Male<br> <input type="radio" name="gender" value="female"> Female<br> <input type="radio" name="gender" value="other"> Other </form> </body> </html>

Submitting Forms:

Using, input type="submit", will create a button for submitting the form data to a form handler. The form-handler is typically a server page with a script for processing input data. The form-handler is specified in the form's action attribute:

The action attribute defines the action to be performed when the form is submitted. Normally, the form data is sent to a web page on the server when the user clicks on the submit button. To further understand actions one must first have a basic understanding of PHP.
