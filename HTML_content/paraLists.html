Paragraphs and Lists:

One of the most important HTML5 features and also one of the simplest is the paragraph element.
The paragraph element allows text to be displayed on a webpage that automatically adjusts to the window size.
The page break element can be used to create new lines inside of a paragraph, since whitespace is automatically ommited.

<!DOCTYPE html> <html> <head> <meta charset = "utf-8"> <title> Understanding Whitespace </title> </head> <body> <h1> Paragraph one without a break element. </h1> <p> This is the first line of text. This is the second line of text. This is the third line of text. </p> <h1>Paragraph two without a break element</h1> <p> This is the first line of text.<br> This is the second line of text.<br> This is the third line of text. </p> <body/> <html/> Running the above code gives us:

Lists:

Lists are an equally important HTML concept. They allow us to present organized information and are the foundation for website navigation systems.
Lists can be numbered using the "ol" tag or bulleted using the "ul" tag. The first stands for ordered list and the second unordered list.

<!DOCTYPE html> <html> <body> <h2>An unordered HTML list</h2> <ul> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ul> <h2>An ordered HTML list</h2> <ol> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ol> </body> </html>
