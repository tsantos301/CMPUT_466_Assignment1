Height and Width Properties:

The height and width properties are used to set the height and width of an element.

The height and width can be set to auto (this is default. Means that the browser calculates the height and width), or be specified in length values, like px, cm, etc., or in percent (%) of the containing block.
For example: The element below has a height of 200 pixels and a width of 50% of the container width.

<style> div { height: 200px; width: 50%; background-color: powderblue; } </style>

Setting the maximum width of an element:

The main use of max width is when the browser window is actually smaller than the width of the element.
The browser then adds a horizontal scrollbar to the page for the user to view the element.
Using max-width instead, in this situation, will improve the browser's handling of small windows.

div { max-width: 1000px; height: 200px; background-color: blue; }
