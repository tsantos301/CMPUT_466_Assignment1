Fonts:

The CSS font properties define the font family, boldness, size, and the style of a text.

Font Family:

Font family is used to specify the custom font that is to be used. A link must be specified in the HTML header for a custom font so that the page knows where to find it.

<head> <link href="https://fonts.googleapis.com/css?family=Permanent+Marker" rel="stylesheet"> <style> #specialHead{ font-family: 'Permanent Marker', cursive; } #specialBody{ font-family: 'Amatic SC', cursive; } </style> </head>
