Syntax:

A CSS rule-set consists of a selector and a declaration block:

Selector {property:value; property:value; ... property:value;}

An example would be:

h1 {color:green;}

This would cause all level 1 headings to be the color green unless overridden elsewhere in the CSS document.

In the above case we selected to style all common HTML elements. We can however style by specific element ID, object class and even group selectors.

Selector Overview:

  • The element selector is: elementName { }
  • The specific element ID selector is: #elementID {}
  • The class selector is: .className {}
  • Finally grouping selectors is done using commas: h1, h2, p, body {}
