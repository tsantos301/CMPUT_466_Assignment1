Margins and Padding:

Margins:

The CSS margin properties are used to create space around elements, outside of any defined borders.

With CSS, you have full control over the margins. There are properties for setting the margin for each side of an element (top, right, bottom, and left).


<!DOCTYPE html> <html> <head> <style> #changedMargins{ border: 1px solid black; margin-top: 300px; margin-bottom: 300px; margin-right: 100px; margin-left: 0px; background-color: lightyellow; } div { border: 1px solid black; margin-top: 100px; margin-bottom: 100px; margin-right: 150px; margin-left: 80px; background-color: lightgreen; } </style> </head> <body> <h2 style="text-align: center;">Adjusting Margins of divisions to Illustrate Margin Properties</h2> <div>This is the first Division</div> <div id="changedMargins">This is the second Division with different margin styling.</div> </body> </html>

Margin properties Overview:

  • auto - the browser calculates the margin
  • length {} - specifies a margin in px, pt, cm, etc.
  • % - specifies a margin in % of the width of the containing element
  • inherit - specifies that the margin should be inherited from the parent element

Padding:

The CSS padding properties are used to create space around an elements content, outside of any defined borders.

With CSS, you have full control over padding. There are properties for setting the padding for each side of an element (top, right, bottom, and left).

Note: The padding properties are the same as the Margin properties above with the exception of no "auto" property.

<!DOCTYPE html> <html> <head> <style> #changedPadding{ border: 1px solid black; padding-top: 300px; padding-bottom: 300px; padding-right: 100px; padding-left: 0px; background-color: lightyellow; } div { border: 1px solid black; padding-top: 100px; padding-bottom: 100px; padding-right: 150px; padding-left: 80px; background-color: lightgreen; } </style> </head> <body> <h2 style="text-align: center;">Adjusting Padding of divisions to Illustrate Padding Properties</h2> <div>This is the first Division</div> <div id="changedPadding">This is the second Division with different margin styling.</div> </body> </html>
