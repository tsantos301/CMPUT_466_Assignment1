Introduction:

This is a primer in a series of tutorials that will calmly introduce you to the very basics of Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media and saves a lot of work. It can control the layout of multiple web pages all at once. Eventually, you'll easily know enough to style your own HTML web pages.

The following are the main sections and ideas for this tutorial:

  • Syntax - easing into the language.
  • Backgrounds - changing the background of a web page
  • Margins and Padding - adjusting indentation and border distance
  • Height and Width - adjusting the height and width of elements
  • Boxing Methodology - intro to some page essentials
  • Fonts - Importing special fonts into a web page
